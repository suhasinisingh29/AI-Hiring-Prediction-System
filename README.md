# 🤖 AI Hiring Prediction System

## 📌 Overview

The AI Hiring Prediction System is a Machine Learning web application that predicts whether a candidate is likely to be **Hired** or **Rejected** based on candidate information.

The application is built using **Python**, **Streamlit**, and a **Random Forest Classifier**.

---

## ✨ Features

- Candidate information form
- Hiring prediction using Machine Learning
- Prediction confidence score
- Candidate rating
- PDF report generation
- Interactive dashboard
- Data visualizations using Plotly
- Responsive Streamlit interface

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-learn
- Plotly
- Joblib
- ReportLab

## 🚀 How to Run

1. Install the required packages:

```bash
pip install -r requirements.txt
```

2. Run the application:

```bash
python -m streamlit run app.py
```
